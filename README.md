- **KAIST Maker Campus-A-thon**에 디자이너 팀원과 참여 후 기계공학과 팀원 한 명 영입

- 사용 기술 : Arduino, Raspberry Pi, flask
- Retrospect
  - 실제로 실생활에서 느꼈던 문제를 구체화하여 프로젝트 진행
